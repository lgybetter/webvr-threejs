SDEFのままでポニーを1本化、剛体とボーンの位置関係大きさ調整がまだまだ足りない為　大きな動作の時崩れます。

目の形状をモデリングで改変してあるため表情におかしな部分があるかもしれません

ミク・アペンド改変

Tda_miku_v1_05sw.pmx　青水着にヘッドセットブレスレット、アンクレット　体型はアペミクとほとんど変わりないモデル比較用に製作したもの
			ウェイトは、ほぼ初期状態

Tda_miku_v1_typeH.pmx アペンド・ルカと同じモーフ追加テクスチャ改変



各パーツ　ヘッドセット、ブレスレット、アンクレットに消しモーフが付いています。
アペンドタイプは腰パーツと頭パーツに消しモーフ　腰消１（縮小）腰消２（透明化）を同時に使うのが良いでしょう
ヘッドセットのマイクは別パーツになっているのでPMDエディタの知識がある方なら調整が可能
ブレスレットとアンクレット、ヘッドセットにはボーンが入ってるので　モーフィングで大きさ調整、ボーンで位置調整が可能です。
PMDエディタのTransform Viewを使うなら大きさを-値で設定すれば大きくなります。

MMD初心者でモデルイジリもほぼ初心者なので　いろいろおかしな所もあります。
製作者TDA様のモデルなので多少ヘタレが改造しても絵的にはちゃんと見えるかもしれません
TDA様のモデルに出会えた事に感謝します。
